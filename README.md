# Anbulance
### Discover and save wounded animals around your area.

Anbulance allows users to discover and share wounded animals. Wounded animals, shelters and veterinary clinics are displayed on the map. The user can get directions to the wounded animalsâ€™ location and/or call a nearby shelters through the app.
